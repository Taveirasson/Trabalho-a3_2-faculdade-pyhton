
Utilizar o banco de dados disponibilizado pelo curso de veterinária para montar um dashboard envolvendo gráficos, arquivo excel e banco de dados.

Para usar este projeto é necessario ter instalado MySQL

As bibliotecas usadas são: pandas, Flask, plotly e kaleido, openpyxl, BytesIO, io, mysql.connector

pip install pandas
pip install Flask 
pip install plotly
pip install kaleido
pip install openpyxl
pip install mysql-connector-python

ou 

pip install pandas Flask plotly kaleido openpyxl mysql-connector-python
